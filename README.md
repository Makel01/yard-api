# The Yard API — HBCU Opportunities Directory

## 📌 Overview
The Yard API is a Spring Boot REST API that helps HBCU students discover opportunities such as scholarships, internships, organizations, and events.

This project provides a read-only API that supports browsing and filtering opportunities using HTTP GET requests.

---

## 🛠️ Tech Stack
- Java 17
- Spring Boot 3.x
- Maven
- SLF4J (logging)
- MockMvc (testing)

---

## 📦 Prerequisites
- Java JDK 17+
- VS Code or IntelliJ
- Git (optional)
- Postman (for testing)
